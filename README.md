# Платформа для HR-сотрудников и руководителей для обратной связи и взаимодействия с персоналом

## Описание:

Система автоматизирует взаимодействие между персоналом и HR-командой. Позволяет собирать обратную связь, рассчитывать KPI, внедрять геймификацию и формировать отчеты по эффективности сотрудников.

## Стек технологий

Frontend: React.js
Backend: Node.js

## Архитектура

Проект построен по принципу Layered Architecture (Клиент-Сервер).
Клиент: Веб-интерфейс (адаптивный).
Сервер: REST API + бизнес-логика.
БД: Реляционное хранение данных.
